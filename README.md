# A Simple Queue Server

## Installation
Add the JSON below to your composer.json file and run ```composer update``` to install and update your dependencies.
```json
{
    {
    "repositories": [
        {
            "type": "vcs",
            "url":  "https://github.com/cossay/queue"
        }
    ],
    "require": {
        "cosman/queue": "dev-master"
    }
}
}
```
